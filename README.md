# BitcoinPrice

> Sample Next.js app which shows the current Bitcoin value in USD, GBP and EUR

## Quick Start

``` bash
# Install dependencies
yarn add next react react-dom isomorphic-unfetch

# Serve on localhost:3000
yarn run dev

# Build for production
yarn run build
```
